         - ```public partial class HDevelopExport```

             - ```public HTuple hv_ExpDefaultWinHandle;```

             -  ```public void RunHalcon(HTuple Window)```run函数:输入```Window```,调用action()
             -  ```public void HDevelopStop()```HDevelopStop()函数:输出```MessageBox
